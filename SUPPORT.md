# Support

## What This Project Can Support

This repository is meant to support:

- questions about the documented workflow
- requests for clarification on project status
- discussion of staged validation logic
- collaboration offers from qualified researchers or labs

## What This Project Cannot Support

This repository should not be used for:

- emergency lab or safety support
- legal advice
- investment advice
- claims that bypass the documented evidence boundary

## Best Use Of Support Requests

Good support requests are specific and scoped, such as:

- a documentation gap
- a reproducibility question
- a concern about a scientific claim
- a collaboration inquiry tied to a defined milestone
