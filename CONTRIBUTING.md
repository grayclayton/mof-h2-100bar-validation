# Contributing

Thanks for considering a contribution.

## What Is Helpful Right Now

The most useful contributions at this stage are:

- workflow review
- reproducibility checks
- documentation cleanup
- issue spotting in methods or assumptions
- careful skepticism about claims and evidence boundaries
- introductions to qualified labs or relevant collaborators

## Contribution Rules

Please keep contributions aligned with the current stage of the project.

- do not overstate experimental status
- do not submit unsafe lab guidance outside qualified-lab context
- do not treat modeled outputs as physical proof
- prefer small, reviewable changes over broad rewrites

## How To Contribute

1. open an issue describing the problem or suggestion
2. explain whether the change affects methods, claims, documentation, or code
3. keep pull requests narrow and easy to review
4. include reasoning when you challenge a scientific or technical assumption

## Preferred Standards

- clear writing
- reproducible steps
- explicit uncertainty
- honest handling of negative results
