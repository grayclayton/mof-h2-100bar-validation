# License Decision

## Selected License

This public package is set to Apache-2.0.

## Why Apache-2.0 Fits

Apache-2.0 is a practical choice for a public technical project because it:

- is widely recognized
- allows broad reuse
- includes explicit patent language
- works well for mixed code and documentation repositories

## Next Step

When you publish the repository, keep the top-level `LICENSE` file with the full Apache-2.0 text.
