# Staged Validation Plan

## Objective

Validate hydrogen-storage performance for a base MOF and selected modified variants using a controlled, repeatable external test process.

## Candidate Set

- base MOF control
- V009 scale 1.2600
- V010 scale 1.2600 plus partial Cu/Ni substitution

## Why The Validation Matters

The experimental work is intended to answer a narrow question: do the modeled improvements survive contact with real synthesis, real structure checks, and first-pass adsorption testing?

## Public Validation Stages

1. synthesis quote and sample-production review
2. PXRD confirmation against expected references
3. activation and sample-readiness check
4. room-temperature adsorption and release test
5. short repeat-cycle confirmation

## Controlled Variables

- instrument and calibration lot
- temperature profile
- pressure ramp profile
- sample-mass loading procedure
- operator checklist completion

## Primary Metrics

- uptake at low pressure
- uptake at high pressure
- deliverable uptake
- gravimetric capacity in wt%
- release percentage

## Acceptance Gates

1. mean wt% greater than or equal to 5.000
2. worst-case wt% greater than or equal to 4.980
3. range wt% less than or equal to 0.050
4. mean release greater than or equal to 94.400%

## Evidence Standard

This validation plan is decision-oriented. The result can be positive, mixed, or negative. Any outcome is useful if the procedure, data capture, and interpretation are documented clearly.
