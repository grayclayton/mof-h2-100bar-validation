# Evidence Summary

## Purpose

This page summarizes the public evidence currently supporting the V010 candidate.

The goal is to show why the project is credible enough for a small external validation step without overstating what has already been proven.

## Current Evidence Boundary

The evidence in this repository should be understood as:

- computationally supported
- method-documented
- externally unvalidated

That means the project has a real evidence package, but it does not yet have physical proof.

## What The Evidence Supports

Based on the current project materials, the following claims are reasonable:

- a structured computational screening and packaging process was used to identify and package a lead hydrogen-storage candidate
- the V010 candidate cleared the internal modeled screening gates used in this project
- the project includes a staged validation plan with defined decision points
- the next scientific step is narrow and testable rather than open-ended

## Key Modeled Signals

The current public summary for V010 includes the following modeled indicators:

- about 5.03 wt% modeled hydrogen uptake at ambient temperature (298.15 K) and 100 bar
- about 94.70% mean release
- stable behavior across a 10-cycle durability rerun
- stable secondary temperature-response checks from 273.15 K to 323.15 K
- no apparent mass-transfer bottleneck in the internal review

See [MODELED_RESULTS_TABLE.md](MODELED_RESULTS_TABLE.md) for a compact tabular summary.

See [PUBLIC_EVIDENCE_PROVENANCE.md](PUBLIC_EVIDENCE_PROVENANCE.md) for the file-level map from public claims to public artifacts.

## What Is Not Yet Proven

The current evidence does not yet prove:

- experimental synthesis success
- PXRD confirmation of the intended framework
- experimental adsorption and release performance
- real-world commercial viability

Those remain questions for external validation.

## Why This Still Matters

At this stage, the value of the evidence is not that it closes the scientific case. Its value is that it narrows the next decision to a manageable go or no-go external test.

That is enough to justify:

- a synthesis quote request
- a limited validation conversation with a qualified lab
- a small milestone-based funding ask

## Next External Validation Step

The immediate next step is to determine whether a qualified lab can synthesize the candidate or a close equivalent under realistic conditions.

If that step succeeds, the next validation sequence is:

1. PXRD confirmation
2. activation and sample-quality review
3. ambient-temperature hydrogen uptake and release testing with a 100 bar endpoint
4. short repeat-cycle confirmation

## Publication Standard

This project should publish negative or mixed physical results as well as positive ones. A disciplined negative result would still be useful evidence about the gap between modeled candidates and experimental reality.
